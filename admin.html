Cara embed YouTube:
1. Buka video YouTube → Klik Share → Embed
2. Copy URL dari src="..."
3. Atau paste link video biasa, otomatis dikonversi
Cara embed TikTok:
1. Buka video TikTok → Klik Share → Embed
2. Copy seluruh kode embed HTML
3. Atau paste link video biasa
Cara embed Instagram:
1. Buka postingan → Klik ... → Embed
2. Copy seluruh kode embed HTML
3. Atau paste link postingan biasa
Cara embed Facebook:
1. Buka postingan → Klik ... → Embed
2. Copy kode iframe atau paste link postingan
